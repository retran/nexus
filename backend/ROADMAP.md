**Authentication Flow**:

```text
1. User ‚Üí nexus.example.com
2. Traefik checks Authelia session ‚Üí Not authenticated
3. Redirect ‚Üí auth.nexus.example.com/login
4. Authelia shows OAuth2 providers (Google, Apple)
5. User picks Google ‚Üí Redirect to Google OAuth
6. Google callback ‚Üí auth.nexus.example.com/callback
7. Authelia creates session in Redis
8. Authelia stores user in PostgreSQL
9. Redirect ‚Üí nexus.example.com with session cookie
10. Gateway reads Remote-User header from Authelia
11. Gateway auto-creates/updates user in database
12. User sees Nexus UI - authenticated! ‚úÖ
```
